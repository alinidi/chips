# Chip List Component

Адаптивный React-компонент для отображения списка чипсов в одну строку.  
Если чипсы не помещаются, они скрываются и становятся доступны через попап.

## Функциональность

- Отображение чипсов **в одну строку** на всю ширину контейнера
- **Адаптивность**: при изменении ширины экрана количество видимых чипсов автоматически пересчитывается
- **Скрытые чипсы** доступны через попап по клику на кнопку `...`
- **Выбор чипсов**: клик по чипсу меняет его состояние (выбран / не выбран)
- **Модульность**: компонент `Chip` можно использовать отдельно
- **Переиспользование**: компонент `ChipsList` принимает массив чипсов через пропсы

## Технологии

- **React** (functional components + hooks)
- **TypeScript**
- **SCSS modules**

## Установка и запуск

1. Клонировать репозиторий
```
git clone https://github.com/your-username/chips-list.git
cd chips-list
```
2. Установить зависимости
```
npm install
```
3. Запустить проект
```
npm run dev
```

4. Открыть http://localhost:5173
