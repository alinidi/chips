# Тестовое задание

Вам необходимо разработать компонент, отображающий список чипсов. Компонент должен удовлетворять следующим требованиям:

## Основные требования:
- **Отображение в одну строку**: ВВсе чипсы должны располагаться в одну строку на всю ширину родительского контейнера. Те чипсы, которые не помещаются в строке, должны быть доступны через попап, который открывается при клике на соответствующую кнопку.
- **Адаптивность**: Список чипсов должен автоматически подстраиваться под изменение ширины экрана.
- **Выбор чипсов**: Пользователь должен иметь возможность выбирать чипс по клику на него.
- **Повторное использование**: Компонент должен быть спроектирован таким образом, чтобы его можно было легко переиспользовать в других частях приложения.
- **Модульность**: Чипс должен иметь возможность быть использованным отдельно, вне списка.
